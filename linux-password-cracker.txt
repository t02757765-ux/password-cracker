LINUX PASSWORD CRACKER - KULLANIM DOKÜMANTASYONU
================================================

GENEL BAKIŞ
-----------
Bu araç, Linux ve Unix tabanlı sistemlerde kullanılan şifre hash'lerini 
kırmak için geliştirilmiş Python tabanlı bir password cracker'dır.

DESTEKLENEN HASH TİPLERİ
------------------------
1. MD5-Crypt ($1$) - Eski Linux sistemleri
2. SHA-256-Crypt ($5$) - Orta düzey güvenlik
3. SHA-512-Crypt ($6$) - Modern Linux dağıtımları (varsayılan)
4. DES-Crypt - Çok eski sistemler
5. Blowfish ($2a$, $2b$, $2y$) - BSD sistemleri

ÖZELLİKLER
----------
✓ Otomatik hash tipi tespiti
✓ Wordlist (sözlük) saldırısı
✓ Brute force (kaba kuvvet) saldırısı
✓ Test hash oluşturma
✓ Çoklu platform desteği (Linux, macOS, BSD)
✓ Progress tracking (ilerleme takibi)

KURULIM
-------
Python 3.6+ gereklidir. Ek paket gerekmez.

Gerekli modüller (standart Python kütüphanesi):
- hashlib
- crypt
- re
- argparse
- itertools

KULLANIM ÖRNEKLERİ
------------------

1. YARDIM MESAJINI GÖRME:
   python3 linux-password-cracker.py --help

2. TEST HASH OLUŞTURMA:
   python3 linux-password-cracker.py --generate-test "sifrem123" --hash-type SHA-512-Crypt

3. WORDLIST SALDIRISI:
   python3 linux-password-cracker.py -H "$6$salt$hash..." -w /path/to/wordlist.txt

4. BRUTE FORCE SALDIRISI:
   python3 linux-password-cracker.py -H "$6$salt$hash..." --brute-force --max-length 6

5. ÖZEL KARAKTER SETİ İLE BRUTE FORCE:
   python3 linux-password-cracker.py -H "$6$salt$hash..." --brute-force \
     --charset "abc123" --max-length 5

PARAMETRELER
------------
-H, --hash          Hedef hash değeri (zorunlu)
-w, --wordlist      Wordlist dosya yolu
-t, --hash-type     Hash tipi (otomatik tespit edilir)
--generate-test     Test için hash oluştur
--brute-force       Brute force saldırısı kullan
--charset           Brute force karakter seti (varsayılan: abc...0123...)
--max-length        Maksimum şifre uzunluğu (varsayılan: 6)
-v, --verbose       Detaylı çıktı

HASH TİPİ ÖRNEKLERİ
-------------------

MD5-Crypt:
$1$rounds=1000$saltsalt$hashedvalue

SHA-256-Crypt:
$5$rounds=5000$saltsalt$hashedvalue

SHA-512-Crypt:
$6$rounds=5000$saltsalt$hashedvalue

DES-Crypt:
ABcdEfGhIjKl

Blowfish:
$2y$12$saltsaltsaltsalt$hashedvaluehashedvaluehashed

PRATİK ÖRNEKLER
---------------

Örnek 1: Basit wordlist saldırısı
----------------------------------
python3 linux-password-cracker.py \
  -H "$6$xyz$abcdef..." \
  -w /usr/share/wordlists/rockyou.txt

Örnek 2: Küçük harf ve rakamlarla brute force
----------------------------------------------
python3 linux-password-cracker.py \
  -H "$6$xyz$abcdef..." \
  --brute-force \
  --charset "abcdefghijklmnopqrstuvwxyz0123456789" \
  --max-length 5

Örnek 3: Test hash oluştur ve kaydet
-------------------------------------
python3 linux-password-cracker.py \
  --generate-test "test123" \
  --hash-type SHA-512-Crypt > test_hash.txt

PERFORMANS İPUÇLARI
-------------------
1. Wordlist saldırıları brute force'tan çok daha hızlıdır
2. Küçük max-length değerleriyle başlayın (4-6)
3. Özel karakter setleri kullanarak arama alanını daraltın
4. Büyük wordlist'ler için progress tracking otomatiktir

SORUN GİDERME
-------------
Problem: "crypt module not found"
Çözüm: Bazı Python kurulumlarında crypt modülü varsayılan gelmez.
       Ubuntu/Debian: sudo apt-get install python3-dev libssl-dev

Problem: Hash tipi tespit edilemiyor
Çözüm: Hash formatının doğru olduğundan emin olun. Örnek formatları yukarıda bulabilirsiniz.

Problem: Bellek hatası
Çözüm: Brute force için max-length değerini düşürün (4-5)
